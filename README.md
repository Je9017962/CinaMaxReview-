<h1>CinaMaxReview</h1>  

<h2>Project Description</h2>  
<p>CinaMaxReview is a movie review web app designed for movie‑goers who want a simple and relaxing place to browse films and read community opinions. Users can explore reviews for the latest and most popular movies, as well as upload and review films they’ve seen, even if the movie isn’t already listed. The goal is to create an easy‑to‑use platform where anyone can share their thoughts and discover new movies through a clean, user‑friendly interface.</p>

<h2>Live Demo</h2>
<p>Netlify Deployment: https://cinamaxreview.netlify.app/</p>

<h2>Planned Features</h2>
<li>Movie List Page: Display movie posters, titles, and ratings in a clean, scrollable layout</li>
<li>Movie Detail Page: Show summaries, cast information, and user‑submitted reviews for each film.</li>
<li>Review Submission Form: Allow users to write and submit their own movie reviews.</li>
<li>User Submissions Page: Let users view a personalized list of all the reviews and movies they have submitted.</li>
<li>Dark & Light Mode Toggle: Provide an accessibility‑friendly theme switcher for comfortable viewing in different environments.</li>

<h2>Current Features Implemented</h2>
<li>Browse a list of movies with poster thumbnails</li>
<li>View detailed movie information</li>
<li>Submit reviews for any movie</li>
<li>Add custom movies not already in the list</li>
<li>View all user‑submitted movies and reviews</li>
<li>Toggle between light and dark mode</li>
<li>Data persistence using localStorage (survives refreshes)</li>

<h2>Tech Stack</h2>
<li>Frontend: React + Vite</li>
<li>Styling: Custom CSS / CSS Modules</li>
<li>Deployment: Netlify</li>
<li>Version Control: Git & GitHub</li>
<li>AI Tools Used: Claude (transcript included in Canvas submission) </li>

<h2>Setup Instructions</h2>
<p>1. Clone the repository: git clone git clone https://github.com/Je9017962/CinaMaxReview-.git</p>
<p>2. Navigate into the projet folder: cd CinaMaxReview-</p>
<p>3. Install dependencies: npm install </p>
<p>4. Start the developmetn server: npm run dev</p>

<h2>Known Bugs or Limitations</h2>
<li>Reviews cannot currently be edited after submission.</li>
<li>No backend authentication or cloud database.</li>
<li>Movie search is limited to the static dataset and user submissions.</li>
<li>No advanced filtering or sorting options yet.</li>
